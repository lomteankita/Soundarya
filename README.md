# soundarya

Soundarya is a clone of an e-commerce website Bath & Body Works which sells soaps, lotions, fragrances, and candles.

## Pages and features
- Home Page:- Navigation Bar, Footer,Login,Signup.
- Product Page:- It has sorting & filtering functionality.
- Single Page
- Admin Page :- Admin dashboard.
